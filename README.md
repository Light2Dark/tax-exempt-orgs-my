# Tax-exempted Organizations in Malaysia

## Overview

This project is a web scraper that scrapes the tax-exempted organizations in Malaysia from the [Hasil website](https://www.hasil.gov.my/en/quick-links/services/donation-approval/).

## Usage

```bash
python pipeline.py
```
